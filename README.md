# Hotel Menu Program (Java)

This is a menu-driven Java program created using **do-while loop** and **switch-case** statements.

## Description
The program displays a hotel menu with Veg and Non-Veg options.  
The user can select items repeatedly until choosing the Exit option.

## Concepts Used
- do-while loop
- switch-case
- Nested switch
- Scanner class
- Package in Java

## How to Run
1. Open the program in Eclipse
2. Run the `Dowhile.java` file
3. Enter choices as shown in the menu
